## Project Overview

**MB3DE** is a 3D side scroller rhythm musical game built with Unity using the Built-In Render Pipeline.

### Key Technical Details
- **Engine**: Unity
- **Render Pipeline**: Built-In Render Pipeline
- **Genre**: 3D Side Scroller + Rhythm Game
- **Gameplay Style**: Musical/Rhythm-based
